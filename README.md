# Immanuel Capital Partners — Website

Corporate website for Immanuel Capital Partners Limited, a business and financial advisory firm based in Abuja, Nigeria.

## Structure

This is a static, single-file site — no build step required.

- `index.html` — the entire site (HTML, CSS, and JS in one file)

## Local preview

Just open `index.html` directly in a browser, or serve it locally:

```bash
npx serve .
```

## Deployment

Deployed via Vercel. Any push to `main` redeploys automatically once connected.
